This project is a frontend-only E-Commerce Product Page developed as a final project task for the DDW platform.  
It is built using HTML, CSS, and Vanilla JavaScript without using any frameworks or backend technologies.

The project demonstrates understanding of UI layout, DOM manipulation, JavaScript logic, and localStorage usage.

---

Project Objective

The main objective of this project is to:

- Build a simple e-commerce product page
- Demonstrate frontend development skills
- Use JavaScript for dynamic product handling and cart functionality
- Create a responsive and user-friendly UI

Page Layout

The website includes:

- Header with logo/title
- Navigation bar
- Product listing section
- Cart section
- Footer

The layout is responsive and works on both mobile and desktop screens.

---

Product Listing

- Products are displayed dynamically using JavaScript
- Each product includes:
  - Image
  - Name
  - Price
  - Short description
  - Add to Cart button
- Product data is stored in a JavaScript array of objects

---

Cart Functionality

- Add to Cart button adds product to the cart
- Quantity increases if product already exists
- Cart updates without page reload
- Cart displays:
  - Product name
  - Price
  - Quantity
  - Total price per item
  - Remove button
- Grand total updates automatically

---

Local Storage

- Cart data is saved using localStorage
- Cart data is restored on page reload
- Cart is not cleared when the page refreshes
